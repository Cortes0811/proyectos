ZEP Big Orange

Desengrasante Industrial Citrico

FICHA TECNICA

Descripción

Big Orange es una mezcla de desengrasantes, detergentes y emulsionantes orgánicos naturales.que puede usarse como reemplazo de desengrasantes a base de petróleo

Composición

  • >> Forma física: Líquido
  • >> Color: Claro, naranja
  • >> Olor: Naranja (fuerte)
  • >> pH (concentrado): No aplicable
  • >> Punto de inflamabilidad: Vaso cerrado: 49.4 ° C / 120.9 ° F
  • >> Gravedad específica: 0.85
  • >> Duracion: Mínimo 1 año

    • Caracteristicas y Beneficios

  • >> Disolvente de cítricos Formulado con solvente cítrico natural; no contiene destilados de petróleo, disolventes clorados, butil cellosolve, cáusticos o ácidos
  • >> Alta solvencia Poder desengrasante comparable al de los destilados de petróleo y los disolventes clorados
  • >> Fragancia agradable Deja una fragancia cítrica fresca; no tiene un disolvente desagradable ni olor a petróleo
  • >> Versátil Este producto se recomienda para su uso como desengrasante de piezas, equipos y suelos. También se puede utilizar para eliminar grafitis y masilla de una variedad de superficies
  • >> Emulsionable Se emulsiona fácilmente con agua para ahorrar
  • >> Biodegradable Todos los detergentes y emulsionantes de Big Orange son completamente biodegradables; este producto también está libre defosfatos
